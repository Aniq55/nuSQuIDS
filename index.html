Welcome to nuSQuIDS.

nuSQuIDS is a neutrino oscillation software build on SQuIDS (https://github.com/jsalvado/SQuIDS), which was published in Computer Physics Communications (http://www.sciencedirect.com/science/article/pii/S0010465515002672). It has been designed to be a fast and flexible library to calculate neutrino oscillations.
